  - **authors:** Reed Lau
  - **institution:** Tencent
  - **link:** https://arxiv.org/pdf/2606.06521
  - **Simple LLM Summary:** The paper analyzes two implementation choices—KV block iteration order and static scaling factor—that critically affect precision when casting the softmax probability matrix P to FP8 in attention computation under the Attention Sink phenomenon.
